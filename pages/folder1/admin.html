# 一. 引言

# 1. 编写目的

该文档主要介绍CXCC 云呼叫中心平台超级管理员的主要功能，便于指导客户快速使用系统和独立学习参考，该文档主要读者为，平台超级管理员和系统运维人员

# 2. 背景

为帮助CXCC 云呼叫中心平台的代理客户与自建平台客户尽快熟悉系统功能、掌握管理与配置的方式，了解系统的特性，特编CXCC 云呼叫中心平台操作手册，分超级管理员端、管理员端和座席端使用手册，本文档为CXCC 云呼叫中心平台超级管理员端的使用手册。

# 3. 注意事项

  • 推荐使用Chrome 浏览器（谷歌浏览器）最新版。
  • 系统提供一个快速下载链接在系统的登录主页如图1.1，可通过点击该链接快速下载谷歌浏览器。

# 2. 系统登陆/退出

# 1. 登陆

打开浏览器，地址栏输入CXCC 云呼叫中心平台服务器地址，点击回车跳转到如图2.1 所示的登录界面。填写正确的“公司名称”、“用户名”和“密码”后，点击登录按钮，即可登录进入超级管理员系统主页，如图2.2 所示。

【公司名称】： 系统部署时，创建该公司（企业）时所填写的公司名称。

【用户名】：公司（企业）账户创建时，默认登录名为“admin”

【密码】：系统部署时，超级管理员默认密码。

# 客户管理

# 5.1客户列表

# 5.1.1 查询

通过“客户列表”菜单进入到客户列表界面后，可以查询到全部的客户信息，如下图5.1 所示，也可以通过过滤条件进行信息筛选。

​ 5.1

# 5.1.2 新增

点击“新增”按钮可以添加新的客户，添加信息界面如下图 5.2 所示。

​ 5.2

【客户名称】创建客户公司（企业）时所填写的公司名称，作为登录时用的“公司名称”。

【密码】创建客户时，默认生成一个用户名为“admin”的管理员，这个密码为 admin 账号登录密码。

【权限模板】设置该用户的模板信息，控制角色权限，角色具体信息操作参考 4.2。

【联系人】客户联系人名称。

【手机】客户联系电话。

【永久时限】设置过期时间，如果关闭后，需要选择【过期时间】，过期后，无法正常进行登录。

【是否为超级管理员】如需创建二级超级管理账户，需要选择此选项。该账户需要拥有超级管理员或超级管理员克隆的权限模板才有效。

【状态】客户账号使用状态，锁定后无法正常登录系统。

【备注】备注客户信息。

填写完成后，点击“确定”按钮即可创建一个新的客户。

# 5.1.3 充值

点击用户列表操作中的“充值”按钮，即可进行话费充值，如需要扣款，填写相应负数即可。

# 5.1.4 编辑

点击用户列表操作中的“编辑”按钮，进入编辑客户信息界面，字段信息参考 5.1.2， 修改完成提交信息，即可完成客户信息编辑。

# 5.1.5 跳转

点击用户列表操作中的“跳转”按钮，即可以客户 admin 账号身份直接跳转进客户管理系统。

# 5.2坐席列表

通过“坐席列表”菜单进入到坐席列表界面后，可以查询到全部的坐席信息，如下图5.3所示，也可以通过过滤条件进行信息筛选

​ 5.3

【分机号】创建坐席时，系统用户名与分机号进行绑定，作为坐席登录用户名和 sip 软电话注册号码。

【姓名】坐席姓名设置，默认与分机号同名。

【认证密码】坐席登录系统密码，坐席注册 sip 软电话认证密码。

【客户】分机号所属客户账号，登录时作为【公司名称】。

【分级状态】显示分机是否正常注册。

【注册 IP】分机注册成功后，显示真实注册 IP 地址信息。

# 5.2.1 新增

点击“批量添加”按钮，添加新的坐席工号，添加界面如下图 5.4 所示。信息填写完成后，点击“确定”按钮即可完成坐席工号添加。

​ 5.4

注：分机号无法重复，如开始分机号至结束分机号中有工号已被添加，无法正常添加工号。

# 5.2.2 批量删除

选中需要删除的坐席工号，点击“批量删除”按钮，即可完成删除操作，如下图 5.5 所示。

​ 5.5

# 5.2.3 修改密码

选中需要修改面的坐席工号，点击“修改密码”按钮，进入编辑密码界面，输入新的密码后提交信息，即可修改坐席密码。

# 5.2.4 编辑

点击坐席列表操作列下的“编辑”按钮，可进行坐席姓名编辑。

# 5.3中继落地

通过“中继落地”菜单进入到坐席列表界面后，可以查询到全部的中继信息，如下图

5.6 所示，也可以通过过滤条件进行信息筛选。

图 5.6

【注册状态】显示注册模式的中继的注册状态。

# 5.3.1 新增

点击“新增”按钮，添加新的中继线路，添加界面如下图 5.7 所示。信息填写完成后， 点击“确定”按钮即可完成中继号码添加。

图 5.7

【中继名称】中继命名。

【对接模式】“注册”模式使用下列参数信息中的【SIP 号码】与【密码】两个信息进行线路注册；“IP 对接”使用下列参数信息中的【主叫前缀】与【被叫前缀】两个信息进行规制送号。

【SIP 号码】注册分机电话号码。

【密码】注册分机密码。

【主叫前缀】送号信息中的主叫前缀信息。

【被叫前缀】送号信息中的被叫前缀信息。

【域名】注册 IP，送号 IP。

【端口】注册端口，送号端口。

【并发上限】此中继最大并发数。

【优先级】此中继在路由下的呼叫优先级，数值越大，优先级越小。

【outboundProxy】IMS 注册所需参数 outboundProxy。

【registerProxy】IMS 注册所需参数 registerProxy。

【from-user】IMS 注册所需参数 from-user。

【from-domain】IMS 注册所需参数 from-domain。

【注册周期】IMS 号码注册周期时长

【计费费率】通话收费费率，费率规则见下文 5.8。

【语音编码】选择线路允许的语音编码。

【DTMF 接受】DTMF 信息接收设置。

【caller-id-in-from】主席作为主叫功能。

【客户】选择该中继所属客户。

【线路备注】备注信息。

填写完成后，提交信息，即可完成中继新增。

选中需要删除的中继，点击“删除”按钮，即可完成删除操作，如下图 5.8 所示。

# 5.3.3编辑

点击中继列表操作中的“修改”按钮，进入编辑中继信息界面，字段信息参考 5.3.1 中的信息字段。修改信息填写完成后，提交信息，即可完成中继信息编辑。

# 5.4消费统计

通过“消费统计”菜单进入到消费统计界面后，可以查询到全部用户的消费统计，如下图 5.9 所示，也可以通过过滤条件进行信息筛选，同时可以导出搜索结果。

# 5.5语音审核

通过“语音审核”菜单进入到语音审核界面后，可以查询到全部用户的提交的语音审核信息，如下图 5.10 所示，也可以通过过滤条件进行信息筛选，同时在线播放上传语音与下载。

图 5.10

【审核状态】当前语音文件审核状态，默认为“未审核”。

通过点击操作列表的“审核”按钮，可以进行状态变更，有“通过”和“不通过”两种状态选择，提交审核备注。

# 5.5.2 删除

通过选择需要删除的语音文件，点击“删除”按钮即可删除对应的语音文件，如下图5.11 所示。

​图 5.11

# 5.7 通话清单

图 5.13

图 5.17

# 5.9.2修改中级组

# 5.9.3删除中级组

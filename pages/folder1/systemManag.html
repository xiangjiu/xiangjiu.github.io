# 系统管理

# 7.1角色设置

“角色设置”下，显示拥有的所有用户账号信息，如图 7.1 所示。也可通过筛选条件进行准确查询。

​ 图 7.1

# 7.1.1 新增

点击“新增”按钮可以添加新的登录用户。新增用户界面如图 7.2 所示。新增用户时， 需要填写状态，如在“锁定”状态下，无法登录该平台。

​ 图 7.2

# 7.1.2删除

删除账号时，先选中需要删除的账号，然后再点击“删除”按钮如图 7.3，即可成功删除。

图 7.3

注：系统默认的用户名 admin 用户无法删除！

# 7.1.3编辑、查看

点击页面操作列下的“编辑”与“查看”按钮，如图 7.4 所示，即可展示对应操作信息。

​ 图 7.4

点击“编辑”按钮后，进入编辑页面，如图 7.5 所示。

​ 图 7.5

用户名为只读数据，无法修改，其他信息均可根据需求进行修改。

点击“查看”按钮后，进入编辑页面如图 7.6 所示，显示一些用户相关信息。

​ 图 7.6

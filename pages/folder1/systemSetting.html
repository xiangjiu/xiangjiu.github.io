# 系统设置

# 6.1license证书

通过“license 证书”，查询系统的授权信息。

# 6.1.1获取机器码

获取机器码，用于更新授权信息。

# 6.2公告维护

通过“坐席维护”菜单进入到公告维护界面后，可以查询到全部的公告信息，如下图6.1 所示，也可以通过过滤条件进行信息筛选。

​ 图 6.1

# 6.2.1新增

点击“新增”按钮，发布新公告，界面如图 6.2 所示。

​ 图 6.2

【所有对象】如果关闭，需要选择该公告的【公告对象】（用户公司名称）。

【公告对象】选择该公告面向的用户。

【标题】公告标题。

【内容】公告内容。

【过期时间】公告展示过期时间。

# 6.2.2删除

选择需要删除的公告，点击“删除”按钮，即可完成删除。如图 6.3 所示。

​ 图 6.3

# 6.2.3编辑

点击公告列表操作中的“编辑”按钮，进入编辑公告信息界面，字段信息参考 6.2.1 中的信息字段。修改信息填写完成后，提交信息，即可完成公告信息编辑。

# 6.6客户参数

创建客户后，会自动生成该客户的客户参数。在客户参数中，灵活调整客户需要的独立配置。客户参数界面如下图 6.7 所示。

​ 图 6.7

# 6.6.1编辑

点击客户参数列表操作栏的“编辑”按钮，即可编辑客户参数，界面如下图 6.8 所示。

​ 图 6.8

【最大并发】该公司能达到的最大并发值。

【录音时段】该公司呼叫过程的录音时机。

【录音命名格式】选择录音命名参数排序（date 必填）。

【录音文件格式】支持 wav 和 mp3 格式，支持单双音轨。

【接通提示音】坐席电话接听提示音。

【获取验证码功能】验证码功能开启关闭控制。

【是否启用 webrtc】开启关闭 webtrc 软电话。

【copyright】系统下方专利展示。

【工作时间内是否开通查询功能】开启关闭工作时间内是否开通查询功能。

【工作时间内是否开通导出，下载功能】开启关闭工作时间内是否开通导出，下载功能。

【工作开始时间】--【工作结束时间】时间段外无法正常呼叫。

【录音保存天数】录音保存天数，超过天数录音文件自动删除。

【数据保存天数】话单、客户资料保存天数，超过天数数据自动删除。

【数据导出量】每次导出限制导出行数。

【查询时间跨度天数】限制系统查询范围。

【振铃时长】呼叫超过振铃时长，主动挂断。

【最低余额提醒】提醒客户管理员系统余额告警的数值。

修改客户参数后，提交后实时生效。
